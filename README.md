📊 Integrated Data Analysis and Visualization Project

Walmart Customer Purchases Dataset Analysis

An end-to-end Data Analysis and Visualization** project developed using Python,  Pandas ,  NumPy ,  Matplotlib , and  Seaborn. This project demonstrates the complete data science workflow, including data loading, cleaning, exploratory data analysis (EDA), visualization, and interpretation of business insights.

---

📌 Project Overview  

The primary objective of this project is to analyze customer purchasing behavior using a real-world Walmart customer purchases dataset.

The analysis focuses on identifying:

* Customer purchasing trends
* Product category performance
* City-wise sales analysis
* Monthly and quarterly sales trends
* Payment method preferences
* Relationships between numerical variables
* Distribution of purchase amounts
* Business insights through data visualization

This project serves as a complete example of an Exploratory Data Analysis (EDA) workflow suitable for academic submissions, GitHub portfolios, and internship applications.

🚀 How to Run the Project

1. Clone the Repository

```text
git clone https://github.com/YOUR_USERNAME/Integrated-Data-Analysis-Project.git
```
---

2. Open the Project

```text
cd Integrated-Data-Analysis-Project
```
---

3. Create a Virtual Environment (Optional)
```text
python -m venv venv
```
Activate it:

### Windows
```text
venv\Scripts\activate
```

### Linux / macOS
```text
source venv/bin/activate
```
---

4. Install Dependencies

```text
pip install -r requirements.txt
```
---

5. Launch Jupyter Notebook
```text
jupyter notebook
```

Open:
```text
notebook/Integrated_Data_Analysis.ipynb
```

Run all cells to reproduce the analysis and visualizations.
